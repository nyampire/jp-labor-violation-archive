このプロジェクトについて

厚生労働省が公開する「労働基準関係法令違反に係る公表事案」のPDFをアーカイブし、 時系列での掲載状況を追跡するプロジェクトです。

厚労省の公表データは毎月更新され、一定期間後に削除されるため、 過去のデータを参照することが困難です。このリポジトリでは、PDFの原本保存と 掲載期間の追跡を行っています。

データソース

  • 厚生労働省 - 長時間労働削減に向けた取組
  • H-CRISIS（国立保健医療科学院）
  • Internet Archive Wayback Machine
